# Cortex-managed repo

This repo is registered with the Claude Cortex plugin. On session start, Cortex identifies this repo against its registry in the Obsidian vault and loads the relevant project context automatically.

## Why this file is so short

This file is intentionally minimal. Project context, dev preferences, decisions, and changelogs all live in the Obsidian vault. The vault is the single source of truth.

If you need to update project information, edit the corresponding files in the vault — not this file. Cortex will surface the updates automatically on the next session.

## Do not edit by hand

This file is managed by Cortex. Manual edits may be overwritten. To change how Cortex behaves in this repo, update the project's context in the vault.
